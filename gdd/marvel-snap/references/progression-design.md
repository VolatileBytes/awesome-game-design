# Progression Design

## Collection Level (CL)

The primary progression axis. Every player starts at CL 1 and climbs.

### How CL Grows

- **Upgrade a card** = boost = 1–6 CL
- Boosters (generic currency) + Credits (premium-but-earnable) → upgrade a card's visual tier (Common → Uncommon → Rare → Epic → Legendary → Ultra → Infinity)
- Each upgrade rewards CL
- CL is **account-wide**, not per-card

### CL Rewards

- Every **CL threshold** grants a reward:
  - Collector's Reserves (random card, credits, boosters, variants)
  - Fixed rewards at round numbers (new cards, avatars)
  - Cache rewards every ~10 CL

### Pool Unlocks

- **Pool 1**: CL 1–214 (~46 cards) — starter cards
- **Pool 2**: CL 214–474 (~25 cards) — more complex effects
- **Pool 3**: CL 474–1000 (~70 cards) — the broad meta
- **Pool 4**: CL 1000+ (~20 cards rotated as "Spotlight Caches")
- **Pool 5**: elite cards in Spotlights
- **Pool 6**: season pass & newest releases

## Season Pass

Monthly pass, ~$10:
- **New Season Card**: exclusive for ~1 month; rotated into general pool later
- **Variants**: cosmetic card alt-arts
- **Credits + Gold** for progression
- **Avatar + Title** cosmetics
- **Free track**: some content, but limited

## Infinity Ladder

Ranked climb:
- Ranks 0–100 map to icons (Iron, Bronze, Silver, Gold, Platinum, Diamond, Vibranium, Infinite)
- Climb via **cubes** (from winning / losing + Snap multipliers)
- End-of-season rewards based on highest rank
- **Infinite** = the top; resets each season but retains a badge

## Conquest (Competitive Mode)

- **Bo3 knockout** tournaments
- Tiers: Bronze → Silver → Gold → Infinite
- Entry tickets gate access; rewards scale with tier
- Competitive-flavored mode for players who want formal structure

## Mastery (per card)

- Mission-style goals per card
- e.g. "Win 100 games with this card in deck"
- Cosmetic medals + avatar + flex

## Events

- **Bi-weekly themes**: "Destroy Week" (bonus for destroy decks), "Move Week", etc.
- Event missions reward **Credits + Gold**
- Special variants earnable in events

## Onboarding

- **Tutorial**: 10 minutes, teaches lanes, mana, priority, Snap
- **Starter Collection**: 30+ pre-unlocked cards to build basic decks
- **Guided matches**: first matches against bots or throttled opponents
- **Daily missions** from day one to build the habit

## Anti-Patterns

- **Pay-to-power**: cards behind paywalls that are stronger than f2p cards
- **Ladder where new cards = wins**: rotation must be skill-first
- **Mandatory new-season-card-play**: if the meta *requires* the paid card, the economy breaks trust
