# Economy

## Currencies

| Currency | Earn | Spend |
|---|---|---|
| **Gold** | Daily / weekly quests, rewards track, modes | Packs, Arena entry |
| **Arcane Dust** | Disenchant cards, rewards | Craft specific cards |
| **Runestones** | Purchase with money | Cosmetics, bundles, event entry |
| **XP** | Playing, quests | Progress rewards track |
| **Event Tokens** | Event-specific earns | Event-specific shop |

## Pack Types

- **Classic / Standard**: generic set packs
- **Expansion Packs**: card drops from specific expansions (most packs)
- **Golden Packs**: all 5 cards guaranteed golden (rare, gifted)
- **Mega Bundles**: pre-order bundles with exclusive cosmetics

## Pack Drop Rates

Roughly (historically published):
- 72% of pack slots: Common
- 22% Rare
- ~4.5% Epic
- ~1.5% Legendary
- Guarantee: 1 Rare+ per pack; 1 Legendary per 10 packs (pity timer)
- Golden (cosmetic) drops at ~5% per card slot, with pity rules

## Card Rarities

- **Common** (white gem): disenchant 5, craft 40
- **Rare** (blue): 20, 100
- **Epic** (purple): 100, 400
- **Legendary** (orange): 400, 1600 — 1 copy only in deck

**Golden** (animated art) doubles the dust values.

## Tavern Pass (Battle Pass)

~$20, seasonal (3 months):
- Bonus XP multiplier
- Exclusive hero skin
- Exclusive card back + coin
- Battlegrounds Perks (choose-2 hero at battleground start)
- Returns value for engaged players

## Arena

- Entry: 150 gold or $2 (Runestones)
- Build a 30-card deck via draft: 3 cards offered, pick 1, 30 times
- Play matches; reward track based on wins before 3 losses
- **Rewards scale**: 7+ wins → break-even+ gold/pack
- **Skill-rewarded mode**: pros go infinite (never spend net gold)

## Mercenaries (Legacy) / Battlegrounds

Battlegrounds is free-to-play for the base mode. Tavern Pass unlocks perks like more hero choices; pay-to-*flex* but not pay-to-win.

## Bundle Shop

- Monthly themed bundles
- Signature Hero skins + card packs + cosmetics
- "Catch-up" bundles for returning players

## F2P Progression Pace

- **~2–3 packs per day** from quests + daily grind
- ~1 pre-order bundle worth of new cards in ~20 days of active play
- Crafting 2–3 key Legendaries per expansion via dust from opening duplicates
- Casual players can build 1 competitive deck per expansion; dedicated f2p 2+

## Dust Economy Management

The dust economy balances pack-opening "luck" with the ability to target specific cards. Blizzard has iterated:

- Duplicate Protection: no dupes of higher-rarity until full collection
- Golden-bump on duplicates: openings more likely to yield gold versions
- Core Set: free starting cards each year

## Monetization Segments

- **Whales**: buy pre-order bundles, Tavern Pass, cosmetics. Maintain full collections.
- **Dolphins**: buy 1 pre-order + Tavern Pass per expansion. ~$40/month heavy months.
- **F2P**: grind quests, Arena, Battlegrounds. Build 1–2 decks per expansion.
- **Returners**: catch-up mechanics + friendly onboarding bring them back cheaply.

## Anti-Patterns

- **Ignoring dust economy**: players hate getting dupe Legendaries without recourse
- **Expansion-power creep**: if every new set demands a whole new deck, churn goes up
- **Cosmetic only = stingy paid track**: Tavern Pass needs some XP/power-adjacent value to be worth buying
- **Runestones only (no gold path)**: cutting gold-buy paths to force cash purchases alienates f2p

## Event Shops

Seasonal events (Fire Festival, Hallow's End):
- Event quests → Event Tokens
- Event Shop: spend tokens on event-specific cosmetics + packs
- Creates a time-limited engagement burst
